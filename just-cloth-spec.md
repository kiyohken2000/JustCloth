# Just Cloth - 仕様書

## アプリ概要

国旗損壊罪をテーマにした政治風刺・教育インタラクティブアート作品。
世界195カ国の国旗を「布として廃棄」する体験を通じて、各国の国旗損壊に関する法律と
表現の自由について考えるvisionOSアプリ。

---

## コンセプト

「布は布です。」

国旗をポリエステルと綿でできた布として扱うことで、
愛国心や国旗を敬えという価値観の押し付けに対する
反抗・憤り・ばかばかしさを体験として表現する。

説教しない。体験させる。

**反抗** → 「禁止」から始め、それでもやらせる。禁じられているからこそやる。

**憤り** → 理不尽な罰則を事実として積み上げる。怒りは自然に生まれる。

**ばかばかしさ** → 廃棄後に「布は布になりました。（もともと布でした）」と淡々と告げる。

## 画面構成

### 起動画面

```
この布を廃棄することは
世界61カ国で犯罪です。

続けますか？
```

「続ける」を選ぶことが最初の反抗になる。
「続けない」を選んだ場合はアプリが閉じる。出口はそこだけ。

続けると世界地図がImmersive Spaceに浮かぶ。
全195カ国がグレーアウトされた状態からスタート。

---

### メイン画面：世界地図

- MapKit for visionOSで世界地図を空間に展開
- 未訪問の国：グレー
- 廃棄済みの国：合法・違法・グレーゾーンで色分け
- 地図の下部に進捗表示

```
14 / 195カ国
```

**色分けルール**

| 色 | 意味 |
|---|---|
| 🟢 緑 | 国旗損壊が合法 |
| 🔴 赤 | 国旗損壊が違法 |
| 🟡 黄 | グレーゾーン・議論中 |
| ⬜ 白 | 法律なし・不明 |

---

### 国の選択

視線で国にフォーカス → ピンチで選択
選んだ国の国旗SVGが空間に大きく展開される

---

### 警告画面（廃棄前）

国旗が表示された後、廃棄方法を選ぶ前に警告を挟む。

**違法な国の場合**

```
警告

[国名]では
この行為は[罰則]に相当します。

それでも廃棄しますか？
```

「それでも廃棄する」を選ばせることで反抗心を引き出す。

**合法な国の場合**

```
[国名]では
この行為は合法です。

つまらないですね。
```

皮肉として機能する。自由があることのほうが「つまらない」と感じさせる逆説。

---

### 廃棄方法の選択

```
この布の廃棄方法を選んでください。

🔥 焼却
✂️ 裁断
♻️ リサイクル
```

視線で選択、ピンチで決定

---

### 廃棄体験

ハンドジェスチャーで廃棄を実行する。

**焼却**
両手で布を持つジェスチャー → 光のエフェクトで布が消えていく
シンプルな光のパーティクル（激しい炎の演出はしない）

```
焼却完了。

布は布になりました。

（もともと布でした）
```

**裁断**
両手で引き裂くジェスチャー → 布が2つに分かれる
断片が空間にゆっくり落ちていく

```
裁断完了。

布は2枚の布になりました。

（もともと布でした）
```

**リサイクル**
布を丸める動作 → 光の粒子になって消える

```
リサイクル完了。

布は別の何かになります。

（もともと布でした）
```

---

### 情報表示

廃棄後、静かに情報が浮かぶ。

**フォーマット**

```
[国名]
人口：[人口]

国旗損壊：[合法 / 違法 / グレーゾーン]

[その国の具体的な法律・判例・エピソード]

[締めの一文]
```

**例：日本**

```
日本国
人口：1億2400万人

国旗損壊：現在合法

しかし2024年、
刑事罰化が議論されました。

提案された罰則：
懲役2年または罰金50万円

今あなたがやったことに。
```

**例：アメリカ**

```
アメリカ合衆国
人口：3億3000万人

国旗損壊：合法

1989年、テキサス州対ジョンソン事件。
国旗を燃やした男性が逮捕された。
最高裁は5対4で「表現の自由」と判断。

以来35年間、合法のままです。
```

**例：ドイツ**

```
ドイツ連邦共和国
人口：8400万人

国旗損壊：違法
罰則：罰金または禁固3年

今あなたがやったことで
逮捕される国です。
```

**最後に必ず表示する素材情報**

```
素材：ポリエステル65% 綿35%
```

### 全カ国制覇エンディング

```
世界の半分以上の国で
あなたは今日やったことが
できます。

布は布です。
```

**statusの値**

## 注意事項

- 国旗SVGはcountry-flagsリポジトリ（MITライセンス）を使用すること
- 各国の法律情報は正確な情報を記載すること（誤情報はアプリの信頼性を損なう）
- エフェクトは激しい炎や破壊表現を避け、シンプルな光のパーティクルにとどめること
- App Store申請時は「教育・政治風刺アート」として申請すること
- TestFlightでの先行配布を経てからApp Store申請することを推奨
